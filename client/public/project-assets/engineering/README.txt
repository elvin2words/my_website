Engineering Artifact Showcase Folder
===================================

Using this folder to publish detailed engineering assets for each project.

Trial structure:
- project-assets/engineering/<project-id>/simulink/
- project-assets/engineering/<project-id>/autocad/
- project-assets/engineering/<project-id>/etap/
- project-assets/engineering/<project-id>/reports/

Trial publish formats:
- Simulink: exported PNG/PDF snapshots + model summary notes
- AutoCAD: exported PNG/PDF sheets (DWG kept as download-only when needed)
- ETAP: one-line diagrams, load-flow and protection coordination reports as PDF

For browser previews, prefer:
- .png, .jpg, .webp, .pdf, .mp4

For native tool files:
- Keep .slx, .dwg, .dxf, .edb, .otb as download links.
